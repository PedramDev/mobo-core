# راهنمای مدیر فروشگاه

## نمای کلی
صفحه «نمای کلی» وضعیت اتصال، لایسنس، همگام‌سازی و صف‌های اصلی را نشان می‌دهد.

## وضعیت‌های همگام‌سازی
- **هماهنگ:** آخرین snapshot پرتال روی ووکامرس اعمال شده است.
- **عقب‌مانده:** محصول باید دوباره از پرتال دریافت و ترمیم شود.
- **در حال ترمیم:** موتور Desired State همان محصول را پردازش می‌کند.
- **خطا:** آخرین تلاش ناموفق بوده و متن خطا باید بررسی شود.

## تاریخ و ساعت
تمام زمان‌های قابل مشاهده در مدیریت افزونه به تقویم شمسی و منطقه زمانی `Asia/Tehran` نمایش داده می‌شوند. مقادیر فنی و صف‌ها در دیتابیس همچنان UTC/Unix باقی می‌مانند.
