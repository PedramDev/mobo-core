شرایط استفاده از MoboCore

آخرین به روز رسانی: ۱۰ جولای ۲۰۲۶

این صفحه شرایط استفاده از سرویس MoboCore در دامنه mobo.codeya.ir را توضیح می دهد. MoboCore برای اتصال فروشگاه های ووکامرس فعال در ایران به جریان محصولات و سفارش های مرتبط با منبع mobomobo.ir طراحی شده است.

۱. معرفی سرویس

MoboCore یک سرویس نرم افزاری برای همگام سازی محصول، تنوع، قیمت، تصویر، وب هوک، صف پردازش، گزارش سلامت و ثبت سفارش های موبویی در فروشگاه های ووکامرس است. افزونه WordPress/WooCommerce با نام Mobo Core از طریق Token به این سرویس متصل می شود.

۲. محدوده استفاده

این سرویس برای فروشگاه های فعال در ایران و workflow مشخص mobomobo.ir طراحی شده است. استفاده از سرویس برای منابع، بازارها یا کشورهایی خارج از این محدوده بدون هماهنگی قبلی پشتیبانی رسمی محسوب نمی شود.

۳. حساب کاربری و دسترسی

برای استفاده از قابلیت های همگام سازی، ثبت سفارش اتوماتیک، وب هوک و گزارش سلامت، مدیر سایت باید یک حساب یا لایسنس فعال در MoboCore داشته باشد و Token اتصال را در افزونه وارد کند. مسئولیت نگهداری محرمانه Token، Webhook Security Code و اطلاعات ورود با مدیر سایت است.

۴. اتصال به سرویس های خارجی

MoboCore ممکن است برای انجام عملیات لازم با این دامنه ها کار کند:

  • mobo.codeya.ir برای مدیریت لایسنس، Token، همگام سازی، صف ها، وب هوک ها، گزارش سلامت و هماهنگی عملیات.
  • mobomobo.ir به عنوان منبع مشخص کالا، بررسی سبد، روش ارسال و ثبت سفارش موبویی در صورت فعال بودن تنظیمات مربوطه.

۵. داده های پردازش شده

بسته به تنظیمات مدیر سایت، داده هایی مانند دامنه سایت، Token، شناسه های محصول و تنوع، عنوان محصول، قیمت، موجودی، تصویر، وضعیت صف، وب هوک، اطلاعات مورد نیاز سفارش، آدرس ارسال، روش ارسال انتخاب شده و گزارش سلامت فنی ممکن است ارسال، دریافت یا پردازش شود.

۶. مسئولیت مدیر سایت

مدیر سایت مسئول این موارد است:

  • وارد کردن اطلاعات اتصال صحیح.
  • تکمیل نگاشت آدرس و روش های ارسال.
  • بررسی صحت قیمت، موجودی، روش ارسال و سفارش قبل از فعال کردن ثبت سفارش اتوماتیک.
  • اجرای Repair کامل بعد از ارتقا از نسخه های قدیمی، مخصوصا نسخه ۷.
  • رعایت قوانین فروش، مالیات، ارسال، پیامک و حریم خصوصی مربوط به کسب و کار خود در ایران.

۷. محدودیت مسئولیت

MoboCore تلاش می کند عملیات همگام سازی و ثبت سفارش را پایدار انجام دهد، اما قطعی اینترنت، اختلال هاست، محدودیت منابع سرور، تغییرات WooCommerce، تغییرات افزونه های دیگر، تغییرات mobomobo.ir یا تنظیمات اشتباه سایت ممکن است باعث خطا یا توقف عملیات شود. مدیر سایت باید قبل از فعال کردن قابلیت های حساس، تست کامل انجام دهد.

۸. پیامک

در صورت فعال سازی پیامک، ارسال واقعی از طریق افزونه «پیامک حرفه ای ووکامرس» و درگاه انتخاب شده در همان افزونه انجام می شود. MoboCore خودش جایگزین درگاه پیامک نیست و تنظیمات درگاه پیامک را مدیریت نمی کند.

۹. تغییرات شرایط

این شرایط ممکن است برای هماهنگی با تغییرات فنی، تجاری یا قانونی به روز رسانی شود. نسخه جدید بعد از انتشار در همین صفحه معتبر است.

۱۰. تماس
